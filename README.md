# DesafioSmartixTech-Back

## Funcionalidades

* Cadastro 
* Login com validação por token
* Logout

## Instalação

Precisa ter [Node.js](https://nodejs.org/) v4+ para rodar.

```sh
$ npm install
```

```sh
$ cd desafioSmartixTech-Back
```

```sh
$ npm run dev
```

## Endpoints

* Cadastro -- /usuarios
* Login -- /login
* Validar token - /login/token
